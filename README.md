# AprendaCode2

## Comandos Git (Terminal)
      
   Configurando usuário e email na máquina local
   ```bash  
   git config --global user.name "SEU_USUARIO"
   git config --global user.email "SEU_EMAIL"
   ```

   Clonando repositório
   ```bash
   git clone https://github.com/Mentyii/AprendaCode2.git
   ```

   Trocar de branch
   ```bash
   git checkout nome_da_branch
   ```
   - branch_tiago
   - branch_matheus
   - branch_leticia
   - branch_gabriela
   - branch_vinicius
   - branch_andre

   Atualizar repositporio LOCAL com base na branch main
   ```bash
   git pull origin main
   ```

   Atualizar repositório REMOTO
   ```bash
   git add .
   git commit -m "descrever_mudança"
   git push origin nome_da_branch
   ```
