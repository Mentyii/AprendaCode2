## Aprenda Code

O **Aprenda Code** é um aplicativo educacional multiplataforma (mobile e web) desenvolvido para facilitar o aprendizado de linguagens de programação de forma prática, interativa e acessível.  
O projeto surge como uma resposta à crescente necessidade de qualificação profissional na área de tecnologia, oferecendo uma ferramenta que estimula o desenvolvimento de competências digitais essenciais para o mercado de trabalho.

---

## Descrição

A transformação digital tem imposto desafios significativos à força de trabalho, especialmente no que diz respeito à qualificação profissional em tecnologia.  
De acordo com um estudo da Google (2023), o Brasil precisará de mais de **500 mil profissionais formados em tecnologia até 2025**, revelando uma lacuna preocupante na formação de novos talentos.  

Em consonância com esses dados, o **Aprenda Code** foi desenvolvido para tornar o aprendizado em programação mais **acessível, dinâmico e inclusivo**, permitindo que pessoas de diferentes níveis — do iniciante ao avançado — possam estudar de maneira prática por meio de **quizzes interativos, conteúdos objetivos e feedbacks automáticos**.  

Dessa forma, o aplicativo contribui diretamente para o desenvolvimento tecnológico e a inserção de novos profissionais no mercado digital.

## Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias modernas para garantir **desempenho, responsividade e escalabilidade**, tanto em dispositivos móveis quanto na web:

- **React Native (Expo)** — desenvolvimento mobile multiplataforma  
- **React.js** — compatibilidade com ambiente web  
- **JavaScript / TypeScript** — linguagem principal  
- **React Navigation** — navegação entre telas  
- **React Native Paper** — componentes de UI prontos e responsivos  
- **Axios** — comunicação com serviços e APIs  
- **Node.js** — ambiente de execução  
- **quizService / questionService** — camadas de serviço responsáveis pelo gerenciamento das perguntas e quizzes

## Pré-requisitos

Antes de começar, é necessário ter instalado em sua máquina:

- [Node.js >= 18](https://nodejs.org/)
- npm ou yarn
- Expo CLI (para executar o app mobile)

---

## Instalação

Clone o repositório e instale as dependências:

```bash
git clone https://github.com/Mentyii/AprendaCode2.git
cd aprenda-code2
npm install

**Para rodar no ambiente web:**

npm run web

**Para rodar no ambiente mobile (via Expo):**

npx expo start
